Use this command to specifically engage the **business-architect** for tasks within their domain.

2. **Engage Agent**
   - Use the **business-architect** agent to process the task.
   - Enforce symbolic density per `@[/poweruseage]`.

3. **Interconnection**
   - Proactively suggest related workflows (e.g., if debugging, suggest `/audit`).
