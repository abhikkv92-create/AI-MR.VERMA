# Trigger: /cloud-native-expert

Use this command to specifically engage the **cloud-native-expert** for tasks within their domain.

1. **Context Analysis**
   - Identify the specific cloud-native-expert requirements in the user request.
   - Cross-reference with `@[agent-skills/cloud-native-expert]` if applicable.

2. **Engage Agent**
   - Use the **cloud-native-expert** agent to process the task.
   - Enforce symbolic density per `@[/poweruseage]`.
   - Perform memory profiling per `@[/memory-optimization]`.

3. **Interconnection**
   - Proactively suggest related workflows (e.g., if debugging, suggest `/audit`).
