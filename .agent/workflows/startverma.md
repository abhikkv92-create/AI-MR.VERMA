This workflow launches the Agent Lightning Local system (Trainer, Collector, Dashboard).

// turbo-all
1. Navigate to project directory
   - Command: `cd e:\ABHINAV\MR.VERMA\agent-lightning-local`

2. Start Docker Containers
   - Command: `docker compose up -d`

3. Verify Status
   - Command: `docker ps`

4. Notify User
   - "System Started Successfully. Dashboard available at http://localhost:8551"
