## Skill Loading

Skills are automatically loaded based on:
1. **Agent frontmatter** — Listed in agent's `skills:` field
2. **Task context** — Detected from user request
3. **Workflow requirements** — Loaded by workflow definition

**Manual loading**: Reference a skill in your request to ensure it's used.
